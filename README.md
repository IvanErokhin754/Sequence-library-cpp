# Sequence Library C++

Учебная библиотека последовательностей на C++, реализующая общий интерфейс `Sequence<T>` с двумя внутренними представлениями:

- на основе динамического массива
- на основе связного списка

Проект также содержит:

- mutable и immutable версии последовательностей
- базовые функциональные операции
- поддержку итераторов
- unit-тесты

## Возможности

Библиотека поддерживает следующие операции:

- получение элемента по индексу
- получение первого и последнего элемента
- получение длины последовательности
- получение подпоследовательности
- конкатенацию последовательностей
- клонирование последовательности
- добавление в конец
- добавление в начало
- вставку по индексу
- удаление по индексу
- отображение (`Map`)
- фильтрацию (`Where`)
- свёртку (`Reduce`)
- поиск первого подходящего элемента (`FindFirst`)

## Архитектура проекта

В основе библиотеки лежит абстрактный интерфейс:

- `Sequence<T>`

От него наследуются две реализации:

- `ArraySequence<T>` — последовательность на основе динамического массива
- `ListSequence<T>` — последовательность на основе связного списка

Для каждой из них существуют две версии:

- `Mutable...` — изменяемая последовательность
- `Immutable...` — неизменяемая последовательность, операции модификации возвращают новый объект

### Схема наследования

```text
Sequence<T>
├── ArraySequence<T>

│   └── ImmutableArraySequence<T>
